# Global Fashion Hub Complete

## Included fields and features
- Full Name
- Mobile / WhatsApp Number
- Email Address
- Company Name
- Country
- Buyer / Supplier / Buyer & Supplier account type
- Password and Confirm Password
- Buyer RFQ form with product, quantity, destination and details
- Consent checkbox
- Product categories and search
- Verified supplier cards
- WhatsApp contact button
- Privacy Policy starter page
- GitHub Pages compatible

## Upload to GitHub
1. Extract the ZIP.
2. Upload `index.html`, `privacy.html`, `assets` folder and `README.md` to the repository root.
3. Open GitHub repository Settings > Pages.
4. Choose Deploy from branch.
5. Select `main` and `/root`, then Save.

## Important
This is a front-end demo. Data is stored only in the browser using localStorage.
For real users, add:
- Secure backend and database
- Password hashing
- OTP or email verification
- Real email notifications
- Admin panel authentication
- HTTPS
- Final Privacy Policy and Terms
